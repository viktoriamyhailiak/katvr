# KateVR Landing Page

KateVR Landing Page is a hard-level, fully responsive landing page designed to showcase the KateVR product, its features, and benefits.
The project follows a Figma design and uses BEM methodology with SCSS, providing a clean, modular, and maintainable code structure.

The layout leverages CSS Grid and Flexbox for a structured, visually appealing, and responsive design, ensuring a seamless user experience across all devices.

Live demo: https://viktoriamyhailiak.github.io/katvr/

Design reference (Figma): [KateVR Figma](https://www.figma.com/file/Blpg4iapsI7fRqJeSp6DvK/KatVR?node-id=1%3A370)

# Technologies Used

HTML5 – semantic structure of the landing page

SCSS (Sass) – modular and maintainable styling

BEM (Block-Element-Modifier) – scalable CSS architecture

CSS Grid & Flexbox – layout for complex sections and responsive design

Responsive Design – optimized for:

- Design: 1440px

- Desktop: 1280px

- Tablet: 640px

- Mobile: >320px

# Setup & Installation

Follow these steps to run the project locally:

1. Clone the repository
git clone https://github.com/viktoriamyhailiak/katevr.git,
cd katevr

2. Install dependencies
npm install or yarn install

3. Run the project locally
npm start or yarn start
